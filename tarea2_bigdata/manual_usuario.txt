MANUAL DE EJECUCIÓN

1. Requisitos:
   - Docker instalado en Windows.
   - Archivos JSON en carpeta `data/`.

2. Construcción de imagen:
   docker build -t spark-supermercado .

3. Ejecución del contenedor:
   docker run -it -v ${PWD}:/app spark-supermercado bash

4. Dentro del contenedor:
   chmod +x run.sh
   ./run.sh

5. Archivos generados:
   - total_productos.csv
   - total_cajas.csv
   - metricas.csv
